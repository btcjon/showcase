## Quest Putt

Interactive mini-golf companion app that layers AI-generated trivia, scoring, and progression over each round.

- Status: active
- Updated: 2026-02-21
- Stack: React, Vite, JavaScript, Anthropic API, Model Viewer
- Impact: Created a hybrid physical+digital game format that makes events more interactive and raises participation time per player.
- Demo: https://questputt.genr8ive.ai/
- Visibility: Core repository is private. This entry summarizes architecture and working approach without exposing proprietary source.

### Problem

Event mini-golf experiences often lose energy between holes and lack personalized engagement mechanics.

### Solution

Engineered a mobile-friendly game layer that injects adaptive trivia and reward progression into each putt sequence.

### Highlights

- Built an AI-trivia gameplay engine that adapts question style by audience and keeps sessions dynamic.
- Blended mini-golf mechanics with digital challenge loops, score progression, and retry-safe state handling.
- Implemented interactive frontend flows optimized for in-person event use with fast-start, low-friction UX.
